# DOMiNO: Diversity Optimization Maintaining Near Optimality

*paper, 2022. Zahavy, Schroecker, Behbahani, Baumli, Flennerhag, Hou, Singh.*

CMDP framing of diverse-near-optimal RL using state-occupancy distance as the inter-policy diversity metric, with explicit hyperparameter control over the quality-diversity trade-off. Concurrent comparator to RSPO; both target the same problem with different mechanisms.
