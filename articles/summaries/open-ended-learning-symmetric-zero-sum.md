# Open-ended Learning in Symmetric Zero-sum Games

*paper, 2019. Balduzzi, Garnelo, Bachrach, Czarnecki, Perolat, Jaderberg, Graepel.*

Geometric "gamescape" framework for nontransitive games where self-play / PSRO fail. Introduces Rectified Nash Response (PSRO_rN), a niching best-response operator that trains only against opponents the new policy beats, producing diverse, broadly-effective populations.
