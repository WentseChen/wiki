# Random Matrix Theory for Deep Learning: Beyond Eigenvalues of Linear Models

*paper, 2025. Zhenyu Liao, Michael W. Mahoney. IEEE Signal Processing Magazine.*

Extends RMT beyond linear model eigenvalues to nonlinear deep networks via the "High-dimensional Equivalent" framework, handling high dimensionality, nonlinearity, and generic eigenspectral functionals. Precisely characterizes scaling laws, double descent, and nonlinear learning dynamics — the theoretical foundation for using spectral metrics (effective rank, singular-value distribution) to analyze fine-tuning and rank collapse.
