# AReaL: A Large-Scale Asynchronous Reinforcement Learning System for Language Reasoning

*project, 2025. Wei Fu, Jiaxuan Gao, Xujie Shen, Chen Zhu, Zhiyu Mei.*

Production-grade RL infra bridging foundation models and agent applications. Fully-async training (boba²) for 2.77× speedup; supports SGLang/vLLM backends; implements 14+ RL algorithms; scales 1.5B–235B with Megatron / FSDP / Archon and NVIDIA + Huawei NPUs.
