# DAPO: An Open-Source LLM Reinforcement Learning System at Scale

*paper, 2025. Yu et al.*

Open-source large-scale RL system for reasoning LLMs that explicitly names entropy collapse as a primary GRPO failure mode and addresses it via four techniques: decoupled clip-higher, dynamic sampling, token-level policy gradient loss, and overlong reward shaping. Achieves 50 points on AIME 2024 with Qwen2.5-32B. Canonical reference linking GRPO entropy/diversity collapse to RL training instability.
