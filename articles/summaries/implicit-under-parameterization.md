# Implicit Under-Parameterization Inhibits Data-Efficient Deep Reinforcement Learning

*paper, 2021. Kumar, Agarwal, Ghosh, Levine.*

Shows that bootstrapped value learning drives the value network's feature matrix toward low effective rank — an "implicit under-parameterization" that hurts offline and online RL data efficiency. Empirics across Atari/Gym plus a linearized proof. The foundational rank-collapse reference the plasticity-loss literature later builds on.
