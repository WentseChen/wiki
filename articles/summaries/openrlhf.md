# OpenRLHF: An Easy-to-use, Scalable and High-performance Agentic RL Framework

*project, 2024. OpenRLHF community.*

Production RLHF framework on Ray + vLLM. Token-in-token-out unified execution; PPO / REINFORCE++ / GRPO / RLOO / DAPO; sync, async, hybrid engine modes; supports 70B+ models, multi-turn agents, VLMs, LoRA. The base codebase MARTI forks.
