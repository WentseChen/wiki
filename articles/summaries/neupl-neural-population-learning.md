# NeuPL: Neural Population Learning

*paper, 2022. Liu, Marris, Hennes, Merel, Heess, Graepel.*

Represents a whole PSRO population with a single conditional network — each policy conditioned on a meta-mixture — and trains all policies concurrently. Avoids re-learning basics and premature truncation; supports cyclic interaction graphs beyond PSRO's scope.
