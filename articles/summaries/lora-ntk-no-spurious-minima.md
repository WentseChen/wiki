# LoRA Training in the NTK Regime has No Spurious Local Minima

*paper, 2024. Uijeong Jang, Jason D. Lee, Ernest K. Ryu. ICML 2024.*

Proves that in the NTK (linearized) regime, LoRA with rank ≳ √N eliminates all spurious local minima, guaranteeing gradient descent finds a low-rank global solution. Gives a concrete rank threshold tied to dataset size and explains why LoRA's constrained optimization succeeds in theory.
