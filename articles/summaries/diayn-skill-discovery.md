# Diversity is All You Need (DIAYN)

*paper, 2019. Eysenbach, Gupta, Ibarz, Levine.*

Unsupervised RL: discover diverse skills with no task reward by maximizing mutual information between a latent skill and visited states under max-entropy actions. Pretrained skills transfer to downstream tasks as initialization or hierarchical options. Canonical reference for MI-based skill discovery.
