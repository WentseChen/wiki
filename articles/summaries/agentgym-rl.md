# AgentGym-RL: Training LLM Agents for Long-Horizon Decision Making through Multi-Turn RL

*project, 2025. Zhiheng Xi, Jixuan Huang, Chenyang Liao, Baodai Huang, Honglin Guo.*

Multi-turn agentic RL framework with decoupled environment / agent / training modules across 5 real-world scenario classes (WebArena, deep search, TextCraft, BabyAI, SciWorld). ScalingInter-RL progressively expands the rollout horizon; 7B model matches GPT-4o on WebArena.
