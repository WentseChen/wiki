# ICM: Curiosity-driven Exploration

*paper, 2017. Pathak, Agrawal, Efros, Darrell.*

Intrinsic curiosity reward = forward-model prediction error in a feature space learned by an inverse dynamics model, so it ignores environment changes the agent can't affect. Canonical intrinsic-motivation baseline; precursor to RND/NGU and trajectory-novelty signals.
