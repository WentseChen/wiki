# Make LoRA Great Again: Boosting LoRA with Adaptive Singular Values and Mixture-of-Experts Optimization Alignment

*paper, 2025. Fan et al. ICML 2025.*

GOAT combines SVD-structured MoE routing over pretrained singular value priors with theoretical optimization-alignment scaling factors to close the LoRA-vs-full-FT gap across 25 datasets. Shows static low-rank subspace selection is a key LoRA bottleneck; adaptive singular value selection via MoE recovers lost capacity.
