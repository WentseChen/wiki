# The Landscape of Agentic Reinforcement Learning for LLMs: A Survey

*paper, 2025. Guibin Zhang, Hejia Geng, Xiaohang Yu, Zhenfei Yin, Zaibin Zhang.*

Comprehensive survey synthesizing 500+ papers on agentic RL for LLMs. Distinguishes single-step MDP (classic RLHF) from extended POMDPs (agentic). Taxonomy across planning / tool use / memory / reasoning / self-improvement / perception, plus a consolidated environment + benchmark map.
