# Modelling Behavioural Diversity for Learning in Open-Ended Games

*paper, 2021. Perez-Nieves, Yang, Slumbers, Mguni, Wen, Wang.*

Defines behavioural diversity geometrically and operationalizes it with a determinantal-point-process (DPP) kernel over policy features. Diverse PSRO regularizes the best-response oracle with this diversity term, lowering exploitability across many open-ended games.
