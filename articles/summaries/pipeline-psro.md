# Pipeline PSRO: A Scalable Approach for Finding Approximate Nash Equilibria in Large Games

*paper, 2020. McAleer, Lanier, Fox, Baldi.*

P2SRO parallelizes PSRO via a hierarchical pipeline of RL workers — each best-responds to the meta-Nash of the levels below — while preserving convergence guarantees. SOTA on Barrage Stratego, scaling linearly with workers.
