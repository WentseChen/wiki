# Extensive-Form Double Oracle (XDO)

Double-oracle algorithm that mixes best-response policies at every infostate of an extensive-form game rather than only at the root (as PSRO does). Guarantees linear convergence in the number of infostates. Neural XDO (NXDO) uses deep RL as the best-response oracle.
