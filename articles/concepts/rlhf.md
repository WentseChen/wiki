# RLHF

Reinforcement Learning from Human Feedback — aligns LLMs by training a reward model on human preference data, then optimizing the language model policy against that reward via PPO or similar RL algorithms. LoRA is widely used to reduce memory requirements for both reward model and policy fine-tuning.
