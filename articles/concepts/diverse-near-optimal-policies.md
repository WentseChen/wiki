# Diverse near-optimal policies

Problem of finding a set of policies that are each near-optimal on the task while being behaviorally distinct from each other — typically formulated as a constrained optimization (maximize diversity s.t. reward ≥ threshold) or via a gated diversity bonus active only above a return floor.
