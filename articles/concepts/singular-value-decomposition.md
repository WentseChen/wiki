# Singular Value Decomposition (SVD)

Matrix factorization W = U Σ V^T decomposing a weight matrix into left singular vectors (U), singular values (Σ), and right singular vectors (V). Used as a diagnostic tool to track how fine-tuning rotates representational subspaces and changes spectral energy distribution.
