# MARTI: A Framework for Multi-Agent LLM Systems Reinforced Training and Inference

**Link:** https://openreview.net/forum?id=E7jZqo0A50

## Summary
Open-source framework for scalable RL of multi-agent LLM systems. Combines centralized multi-agent interactions with distributed per-agent policy training and multi-turn asynchronous rollouts. Multi-agent LLM collaboration outperforms single-agent systems at the same inference budget on math reasoning.

## Key points
- Centralized multi-agent interaction + distributed policy training
- Multi-turn asynchronous rollouts
- Hybrid rewards: rule-based verifiable + LLM-generated
- Built on OpenRLHF; supports debate / MoA / chain workflows
- v2 adds tree-search-augmented multi-agent RL for code reasoning
