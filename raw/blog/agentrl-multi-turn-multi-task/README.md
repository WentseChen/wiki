# AgentRL: Scaling Agentic Reinforcement Learning with a Multi-Turn, Multi-Task Framework

**Link:** https://openreview.net/forum?id=zq3vAmuUk9

## Summary
Open-source agentic RL framework with a fully-asynchronous generation/training pipeline, function-call API, and containerized environments. Introduces cross-policy sampling for multi-turn exploration and task-advantage normalization for multi-task stability; beats GPT-4o / Claude-Sonnet-4 on five tasks.

## Key points
- Fully-async generation-training pipeline
- Function-call API + containerized envs
- Cross-policy sampling for multi-turn exploration
- Task advantage normalization for multi-task RL
- Beats closed baselines on 5 agentic benchmarks
