# Encouraging Divergent Thinking in LLMs through Multi-Agent Debate (MAD)

**Link:** https://github.com/Skytliang/Multi-Agents-Debate

## Summary
Foundational MAS workflow: affirmative/negative agents debate while a judge resolves. Mitigates 'Degeneration of Thoughts' in self-reflection and improves Counterintuitive QA + Commonsense MT. The debate workflow MARTI replicates in run_train_mad.sh.

## Key points
- Names the 'Degeneration of Thoughts' failure
- Affirmative / negative agents + judge
- Improves Counterintuitive QA, Commonsense MT
- Foundational debate workflow for downstream RL methods
