# Training Recipes for Agentic Reinforcement Learning in LLMs: A Survey

**Link:** https://github.com/blacksnail789521/Agentic-RL-Training-Recipes

## Summary
Survey + companion repo systematically organizing agentic-RL training methods: rollout strategies (rejection sampling, self-play), credit assignment (process / outcome rewards), policy optimization (PPO variants, GRPO), plus infra (FP8/FP4, actor-learner decoupling) and env taxonomy.

## Key points
- Rollout: rejection sampling, self-play
- Credit: process vs outcome rewards (PRMs)
- Infra: actor-learner decoupling, FP8/FP4 precision
- Single- vs multi-domain env taxonomy
- Distributed async + curriculum + memory-augmented scaling
