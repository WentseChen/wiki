# verl / HybridFlow: A Flexible and Efficient RL Post-Training Framework

**Link:** https://github.com/verl-project/verl

## Summary
Open RL post-training library built on the HybridFlow programming model; integrates FSDP / Megatron-LM / vLLM / SGLang; supports PPO / GRPO / GSPO / ReMax / RLOO / PRIME / DAPO; scales to 671B with expert parallelism. Backbone for AgentGym-RL, Agent-R1, rLLM, Verlog.

## Key points
- Hybrid-controller decouples data deps from computation
- FSDP / Megatron / vLLM / SGLang integration
- 3D-HybridEngine for efficient actor resharding
- PPO / GRPO / GSPO / ReMax / RLOO / PRIME / DAPO
- Scales to 671B; backbone of many agentic-RL frameworks
