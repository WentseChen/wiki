# GPTSwarm: Language Agents as Optimizable Graphs

**Link:** https://github.com/metauto-ai/GPTSwarm

## Summary
Represents agents and swarms as directed graphs whose edge weights are optimized for task performance via RL and re-prompting. Enables dynamic task routing across specialized agents and self-organizing agent swarms. Workflow precursor cited by MARTI.

## Key points
- Agents/swarms = optimizable directed graphs
- Self-improvement via RL + re-prompting on edge weights
- Modular env / graph exec / LLM backend / memory components
- Dynamic task routing across specialized agents
- Local-LLM compatible (LM Studio)
