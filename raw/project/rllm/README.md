# rLLM: Democratizing Reinforcement Learning for LLMs

**Link:** https://github.com/rllm-org/rllm

## Summary
Framework-agnostic agent-RL: works with LangGraph / SmolAgent / OpenAI SDK by swapping the client. Auto-captures rollouts via a model gateway. Verl + tinker backends with identical APIs. Adds multi-agent training (Solver–Judge joint optimization). 4B beats 235B on finance tasks.

## Key points
- Framework-agnostic via model-gateway client swap
- @rllm.rollout decorator for trace capture
- Verl (multi-GPU) + tinker (single-machine) backends
- Multi-agent Solver–Judge joint training
- 4B model outperforms 235B on finance benchmarks
