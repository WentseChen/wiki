# Reinforcement Learning for LLM-based Multi-Agent Systems through Orchestration Traces

**Link:** https://arxiv.org/abs/2605.02801

## Summary
Surveys and frames RL for LLM-MAS around orchestration traces — temporal interaction graphs covering spawning, delegation, communication, tool use, aggregation, and stopping. Identifies industry/academia scale gap (Kimi Agent Swarm, Codex, Claude Code) and notes no explicit RL methods for stop-decision optimization.

## Key points
- Three axes: reward design / credit signal / orchestration sub-decisions
- Temporal-graph trace schema (replayable JSON)
- Reward families across 8 categories (parallelism, correctness, quality...)
- 84-paper curated pool + 32-record exclusion log
- Highlights stop-decision RL as an open problem
