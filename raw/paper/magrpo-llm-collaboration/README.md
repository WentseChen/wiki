# LLM Collaboration With Multi-Agent Reinforcement Learning

**Link:** https://arxiv.org/abs/2508.04652

## Summary
Frames LLM coordination as a cooperative MARL problem and proposes MAGRPO, removing the need for hand-designed individual rewards. Demonstrates effective collaborative outputs on writing and coding tasks; opens a path to porting other MARL methods to LLM systems.

## Key points
- Frames multi-agent LLM cooperation as cooperative MARL
- MAGRPO extends GRPO with group-level joint optimization
- Avoids brittle individual-reward design
- Validated on collaborative writing + coding
- Open-source code release
