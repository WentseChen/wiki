# Agent-R1: Training Powerful LLM Agents with End-to-End Reinforcement Learning

**Link:** https://arxiv.org/abs/2511.14460

## Summary
Systematizes RL for tool-using LLM agents and ships a modular framework. Extends the MDP formulation specifically for agent components and demonstrates effectiveness on multihop QA. Built on verl.

## Key points
- MDP extension for LLM-agent contexts
- End-to-end RL across tool-use loop
- Modular framework for diverse tasks
- Validated on Multihop QA
