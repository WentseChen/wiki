# Rollout-Training Co-Design for Efficient LLM-Based Multi-Agent Reinforcement Learning

**Link:** https://arxiv.org/abs/2602.09578

## Summary
System framework targeting MAS-RL infrastructure pain points: rollout-training synchronization, rollout load imbalance, and training resource underutilization. Joint orchestrator + asynchronous experience store + hierarchical load balancing + agent-centric resource allocation.

## Key points
- Joint orchestrator across rollout/training
- Async experience store with consistency guarantees
- Hierarchical load balancing for skewed agent loads
- Agent-centric on-demand resource binding
- Up to 7.3× speedup, 5.6× hardware utilization
