# LLM-based Multi-Agent Reinforcement Learning: Current and Future Directions

**Link:** https://arxiv.org/abs/2405.11106

## Summary
One of the first surveys on LLM-based MARL. Lays out cooperative multi-agent settings with shared objectives, open challenges in agent coordination + inter-agent communication, and human-in/on-the-loop architectures unlocked by the language modality.

## Key points
- Single-agent LLM RL frameworks don't readily extend to MAS
- Coordination + inter-agent communication underexplored
- Cooperative shared-objective tasks as primary focus
- Human-in-loop / human-on-loop natively enabled
