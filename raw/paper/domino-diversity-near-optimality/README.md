# DOMiNO: Diversity Optimization Maintaining Near Optimality

**Link:** https://arxiv.org/abs/2205.13521

## Summary

DOMiNO extends the CMDP framing of diverse-near-optimal RL by measuring inter-policy diversity as the distance between state-occupancy distributions rather than via successor-feature correlation. Hyperparameters give explicit user control over the quality-diversity trade-off, and the resulting policies are useful for exploration, transfer, hierarchical control, and robustness.

## Key points
- State-occupancy distance is the diversity metric.
- CMDP framing: near-optimality as a hard constraint.
- Tunable quality-vs-diversity control surface.
- Concurrent comparator to RSPO; both target diverse-near-optimal discovery in 2022.
