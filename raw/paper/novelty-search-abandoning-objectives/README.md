# Abandoning Objectives: Evolution Through the Search for Novelty Alone

**Link:** https://direct.mit.edu/evco/article-abstract/19/2/189/1365

## Summary

The foundational novelty-search paper: replace the fitness/objective function with a reward for behavioral novelty relative to an archive of past behaviors, and search purely for new behaviors. The counterintuitive result — that on deceptive tasks like maze navigation and biped locomotion, novelty search beats objective-driven search — established behavioral diversity as a primary search signal and seeded the entire QD / diverse-policy literature.

## Key points
- Pure novelty reward, no objective signal during search.
- Novelty measured against an archive of past behavior characterizations.
- Outperforms fitness-based search on deceptive problems.
- Conceptual foundation for QD, MAP-Elites, and trajectory-novelty rewards used in RSPO.
