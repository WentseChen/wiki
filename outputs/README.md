# outputs/ — derived artifacts

Q&A renders, slides, images, scratch notes.

- `slides/` — Marp-format `.md` decks. Render via Marp CLI or the Obsidian Marp plugin.
- `images/` — matplotlib / diagrams / screenshots produced from Q&A runs.
- `notes/` — long-form Q&A outputs, research scratch.

Promotable items eventually move back into `articles/` (concept pages, topic overviews) to enrich the compiled wiki.
