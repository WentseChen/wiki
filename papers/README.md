# Papers

Research papers — arxiv, conference proceedings, workshop write-ups. Link only.

## Entry format

```markdown
- [Title](https://arxiv.org/abs/XXXX.XXXXX) — one-line takeaway. *Authors, venue year.* `arxiv:XXXX.XXXXX`
```

Newest at top of the topic file. Drop fields you don't have.
